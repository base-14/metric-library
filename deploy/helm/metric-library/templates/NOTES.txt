To access the application, run:

  # Port forward the web service
  kubectl port-forward svc/{{ include "metric-library.fullname" . }}-web {{ .Values.web.service.port }}:{{ .Values.web.service.port }}

Then open http://localhost:{{ .Values.web.service.port }} in your browser.

To access the API directly:

  kubectl port-forward svc/{{ include "metric-library.fullname" . }}-api {{ .Values.api.service.port }}:{{ .Values.api.service.port }}

API will be available at http://localhost:{{ .Values.api.service.port }}
{{- end }}
